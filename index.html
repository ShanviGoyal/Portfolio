Calculator

Objective
The goal is to create a simple yet functional calculator that works on various devices and screen sizes. The calculator will perform basic arithmetic operations such as addition, subtraction, multiplication, and division.
Features
Responsive Design: The calculator will adjust its layout based on the screen size, ensuring usability on both desktop and mobile devices.
Basic Arithmetic Operations: The calculator will support addition, subtraction, multiplication, and division.
Clear and Delete Functionality: Users will be able to clear the entire input or delete the last entered character.
Keyboard Support: Users can interact with the calculator using both mouse/touch and keyboard input.
Technology Stack
HTML: For the structure of the calculator.
CSS: For styling the calculator and making it responsive.
JavaScript: For the calculator's functionality.
